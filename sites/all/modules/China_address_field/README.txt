INTRODUCTION
------------
China addressfield is based on addressfield. supply a hierarchical select on 
China address like provices, cities, couties etc. Very convenient.

Installlation
------------
1. Copy china_addressfield folder to modules directory (usually sites/all/modules).
2. Download addressfield module
3. At admin/build/modules enable the china addressfield, addressfield, china_address_api 
module in the module lsit.
4. Create an address field, in the field settings, enable the China address addon.
