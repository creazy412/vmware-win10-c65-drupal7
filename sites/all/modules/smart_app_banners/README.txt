 Smart App Banners
=======================

Smart App Banners notify a mobile Safari user when a native app is available (or already installed) for a currently visited site.

Advanced Additional arguments can be passed through the Smart App Banner upon launch allowing an app to respond with a certain View or action.



Instructions
------------------------
1. Install and enable the "Smart App Banners" module.
2. Configure the module by adding your App ID. App ID available via http://itunes.apple.com/linkmaker/
3. Open web site in mobile Safari on an iOS device running iOS 6 or higher.
